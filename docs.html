Как начать

1Откройте личный кабинет вашего API проекта.
2Скопируйте API Base URL и token.
3Вставьте их на странице «Панель» и нажмите «Сохранить».
4Выберите API, действие и отправьте запрос.

Обязательные поля

Category

POST: title, title_en, description, description_en, image

Optional: parent_id

PATCH: status

Post

POST: category_id, title, title_en, description, description_en, image, image_en

Optional: client_id

PATCH: is_published

Comment

POST: post_id, comment, comment_en

Optional: parent_id

GET: post_id нужен в query.
